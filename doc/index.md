## 🔍 About JXAPI

JXAPI generates complete Java SDKs for REST and WebSocket APIs, including:
- Java POJOs
- REST clients
- WebSocket clients
- Rate limiting
- Authentication hooks
- Demo code
- Auto‑generated documentation
